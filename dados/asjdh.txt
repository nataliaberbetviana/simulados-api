Domínio 1: Conceitos da Nuvem (16 questões)

    Pergunta (Escolha 1): Qual dos seguintes é um benefício de usar a nuvem AWS que permite que você pague apenas pelos recursos que consome?
    Opções:
    A. Elasticidade
    B. Economia de escala
    C. Pagamento conforme o uso
    D. Alta disponibilidade
    Resposta correta: C
    Explicação: O modelo "pagamento conforme o uso" (Pay-as-you-go) é um dos principais benefícios da nuvem, eliminando a necessidade de grandes investimentos de capital e permitindo que você pague apenas pelos recursos que utiliza.

    Pergunta (Escolha 1): Uma empresa precisa de uma plataforma para desenvolver e implantar aplicativos sem se preocupar com a infraestrutura subjacente, como servidores e sistemas operacionais. Qual modelo de serviço em nuvem é o mais adequado?
    Opções:
    A. Infraestrutura como Serviço (IaaS)
    B. Plataforma como Serviço (PaaS)
    C. Software como Serviço (SaaS)
    D. Funções como Serviço (FaaS)
    Resposta correta: B
    Explicação: PaaS fornece uma plataforma para os desenvolvedores construírem e implantarem aplicativos, abstraindo a complexidade do gerenciamento da infraestrutura subjacente.

    Pergunta (Escolha 1): Qual modelo de implantação envolve o uso de uma combinação de recursos da nuvem pública e recursos de um data center on-premises?
    Opções:
    A. Nuvem pública
    B. Nuvem privada
    C. Nuvem híbrida
    D. Nuvem de comunidade
    Resposta correta: C
    Explicação: Uma nuvem híbrida é um modelo de implantação que combina uma nuvem pública (como a AWS) com um ambiente de infraestrutura on-premises (nuvem privada), permitindo que os dados e as aplicações sejam compartilhados entre eles.

    Pergunta (Escolha 1): Uma empresa deseja otimizar seus custos na AWS. Qual dos seguintes princípios do pilar de Otimização de Custos do AWS Well-Architected Framework seria mais útil?
    Opções:
    A. Aumentar a utilização de recursos
    B. Adotar um modelo de pagamento por hora
    C. Implementar o monitoramento de segurança contínuo
    D. Usar tipos de instância de uso geral
    Resposta correta: A
    Explicação: O pilar de Otimização de Custos do Well-Architected Framework visa maximizar o valor dos recursos. Aumentar a utilização de recursos é uma forma eficaz de otimizar os custos.

    Pergunta (Escolha 2): Quais dos seguintes são benefícios da computação em nuvem?
    Opções:
    A. Elasticidade
    B. Altos custos de capital (Capex)
    C. Agilidade e inovação
    D. Pagamento por assinatura de software
    Resposta correta: A, C
    Explicação: A elasticidade (capacidade de escalar) e a agilidade e inovação são benefícios da computação em nuvem, enquanto altos custos de capital são característicos de ambientes on-premises.

    Pergunta (Escolha 1): Uma empresa tem uma carga de trabalho que exige que um aplicativo permaneça operacional mesmo que um dos componentes falhe. Qual pilar do AWS Well-Architected Framework trata desse requisito?
    Opções:
    A. Otimização de Custos
    B. Excelência Operacional
    C. Confiabilidade
    D. Segurança
    Resposta correta: C
    Explicação: O pilar de Confiabilidade do AWS Well-Architected Framework garante que um sistema possa se recuperar de falhas e continuar funcionando, o que é um dos principais objetivos da alta disponibilidade.

    Pergunta (Escolha 1): O que é a computação sem servidor (serverless)?
    Opções:
    A. Um modelo onde os servidores são substituídos por contêineres.
    B. Um modelo de nuvem onde você não precisa se preocupar com o provisionamento ou gerenciamento de servidores.
    C. Um serviço que permite criar e gerenciar uma rede privada virtual.
    D. Um tipo de computação de alto desempenho usado para análise de big data.
    Resposta correta: B
    Explicação: A computação sem servidor (serverless) é um modelo de execução de nuvem em que a AWS gerencia a infraestrutura, permitindo que os desenvolvedores foquem no código do aplicativo sem se preocuparem com servidores, escalabilidade ou gerenciamento.

    Pergunta (Escolha 1): Qual é a principal diferença entre um grupo de disponibilidade (Availability Zone) e uma região (Region) da AWS?
    Opções:
    A. Uma região é uma coleção de grupos de disponibilidade isolados.
    B. Um grupo de disponibilidade é uma área geográfica com várias regiões.
    C. Uma região é um único data center, enquanto um grupo de disponibilidade é uma rede de data centers.
    D. Um grupo de disponibilidade é um data center, enquanto uma região é uma área geográfica.
    Resposta correta: A
    Explicação: Uma Região da AWS é uma área geográfica que contém dois ou mais Grupos de Disponibilidade (Availability Zones). Um Grupo de Disponibilidade é um ou mais data centers com energia, rede e conectividade redundantes em uma Região.

    Pergunta (Escolha 2): Quais são os principais benefícios de migrar para a nuvem?
    Opções:
    A. Eliminar completamente a necessidade de planejamento de capacidade.
    B. Requer um investimento inicial significativo em hardware.
    C. Permitir o pagamento apenas pelo que é usado.
    D. Aumentar a velocidade e agilidade da implantação de aplicativos.
    Resposta correta: C, D
    Explicação: A computação em nuvem permite o modelo de pagamento conforme o uso e aumenta a agilidade e a velocidade de implantação, em contraste com a necessidade de planejamento de capacidade e o alto investimento em hardware de ambientes on-premises.

    Pergunta (Escolha 1): Qual termo se refere à capacidade de um sistema de se recuperar de falhas e continuar funcionando?
    Opções:
    A. Elasticidade
    B. Escalabilidade
    C. Confiabilidade
    D. Segurança
    Resposta correta: C
    Explicação: A confiabilidade é a capacidade de um sistema de funcionar corretamente mesmo quando ocorrem falhas, o que é um dos pilares do AWS Well-Architected Framework.

    Pergunta (Escolha 1): O que é o propósito do AWS Global Infrastructure?
    Opções:
    A. Fornecer uma rede de data centers e regiões para hospedar serviços.
    B. Apenas fornecer serviços de armazenamento de dados em todo o mundo.

    D. Oferecer servidores virtuais para aplicativos de alto desempenho.
    Resposta correta: A
    Explicação: A AWS Global Infrastructure é composta por Regiões e Zonas de Disponibilidade interconectadas para fornecer uma infraestrutura escalável, confiável e de alto desempenho para hospedar os serviços e aplicativos de nuvem.

    Pergunta (Escolha 2): Quais são os principais benefícios da computação em nuvem para startups?
    Opções:
    A. Oferece economias de escala.
    B. Requer um alto investimento inicial.
    C. Permite a flexibilidade para crescer.
    D. Exige o gerenciamento completo de servidores e hardware.
    Resposta correta: A, C
    Explicação: A nuvem permite que startups evitem altos custos de capital e se beneficiem de economias de escala e flexibilidade, escalando rapidamente para cima ou para baixo, conforme a demanda.

    Pergunta (Escolha 1): Qual dos seguintes serviços é um exemplo de Infraestrutura como Serviço (IaaS)?
    Opções:
    A. Amazon EC2
    B. Amazon S3
    C. Amazon RDS
    D. Amazon Lambda
    Resposta correta: A
    Explicação: O Amazon EC2 é um exemplo de IaaS, pois oferece instâncias de computação (servidores virtuais), dando aos usuários o controle sobre o sistema operacional e os aplicativos.

    Pergunta (Escolha 1): O que o pilar de Otimização de Custos do AWS Well-Architected Framework visa?
    Opções:
    A. Aumentar o uso de recursos para maximizar a receita.
    B. Manter os custos no mínimo, independentemente do desempenho.
    C. Evitar gastos desnecessários e maximizar o valor dos recursos.
    D. Pagar por licenças de software anuais.
    Resposta correta: C
    Explicação: O objetivo do pilar de Otimização de Custos é evitar gastos excessivos e garantir que a arquitetura da solução esteja alinhada com os objetivos de custo da empresa, através do uso eficiente dos recursos.

    Pergunta (Escolha 1): Qual das seguintes opções não é um benefício de se migrar para a nuvem da AWS?
    Opções:
    A. Oportunidade de economizar custos de capital.
    B. Aumento da velocidade e agilidade.
    C. Possibilidade de implantar globalmente em minutos.
    D. Redução da equipe de TI.
    Resposta correta: D
    Explicação: A migração para a nuvem não necessariamente reduz a equipe de TI, mas sim muda o foco das atividades, de tarefas de manutenção de hardware para o gerenciamento e otimização de serviços em nuvem.

    Pergunta (Escolha 1): O que é um "compute instance"?
    Opções:
    A. Um servidor físico dedicado a um único cliente.
    B. Um servidor virtual na nuvem que você pode usar para executar aplicativos.
    C. Um tipo de banco de dados usado para análise de dados.
    D. Um serviço de armazenamento de objetos.
    Resposta correta: B
    Explicação: Uma instância de computação (compute instance), como no Amazon EC2, é um servidor virtual na nuvem que fornece capacidade de processamento para executar uma variedade de cargas de trabalho.

Domínio 2: Segurança e Conformidade (20 questões)

    Pergunta (Escolha 1): De acordo com o Modelo de Responsabilidade Compartilhada da AWS, quem é responsável pela segurança do sistema operacional convidado, dos aplicativos e das configurações de segurança do firewall?
    Opções:
    A. AWS
    B. O cliente
    C. Um provedor de serviços terceirizado
    D. Ninguém, pois é um recurso automatizado
    Resposta correta: B
    Explicação: No Modelo de Responsabilidade Compartilhada, a AWS é responsável pela segurança "da nuvem" (hardware, software, rede, etc.), enquanto o cliente é responsável pela segurança "na nuvem" (dados, sistema operacional, aplicativos, etc.).

    Pergunta (Escolha 1): Qual serviço da AWS permite que você gerencie o acesso de usuários e permissões de forma centralizada em sua conta AWS?
    Opções:
    A. Amazon EC2
    B. AWS Identity and Access Management (IAM)
    C. Amazon S3
    D. Amazon GuardDuty
    Resposta correta: B
    Explicação: O AWS IAM é o serviço que controla o acesso a recursos da AWS, permitindo que você crie e gerencie usuários e defina permissões para que possam acessar os serviços da AWS de forma segura.

    Pergunta (Escolha 1): Uma empresa precisa garantir que dados sensíveis não sejam acessados por pessoas não autorizadas. Qual princípio de segurança é mais relevante neste caso?
    Opções:
    A. Disponibilidade
    B. Integridade
    C. Confidencialidade
    D. Escalabilidade
    Resposta correta: C
    Explicação: A confidencialidade é o princípio de que a informação só pode ser acessada por indivíduos autorizados, o que é fundamental para proteger dados sensíveis.

    Pergunta (Escolha 2): Quais serviços da AWS podem ser usados para proteger aplicativos web de exploits comuns, como injeção de SQL e ataques de negação de serviço (DDoS)?
    Opções:
    A. AWS WAF
    B. AWS Shield
    C. Amazon RDS
    D. Amazon VPC
    Resposta correta: A, B
    Explicação: O AWS WAF (Web Application Firewall) protege contra injeção de SQL e outros exploits, enquanto o AWS Shield protege contra ataques DDoS.

    Pergunta (Escolha 1): Uma empresa deseja monitorar e registrar todas as ações tomadas por um usuário em sua conta AWS para fins de auditoria. Qual serviço AWS deve ser usado?
    Opções:
    A. AWS Trusted Advisor

    Explicação: O AWS CloudTrail registra as chamadas da API e as ações de usuários em sua conta, fornecendo um histórico de atividades para auditoria e governança.

    Pergunta (Escolha 1): De acordo com o Modelo de Responsabilidade Compartilhada da AWS, quem é responsável pela segurança "da nuvem"?
    Opções:
    A. O cliente
    B. A AWS
    C. Ambos, o cliente e a AWS, em uma parceria 50/50
    D. Um consultor de segurança independente
    Resposta correta: B
    Explicação: A segurança "da nuvem" é responsabilidade da AWS e inclui a segurança da infraestrutura física, hardware, software, rede e instalações que executam os serviços da AWS.

    Pergunta (Escolha 1): Qual serviço permite que você analise vulnerabilidades e desvios de conformidade em suas instâncias do Amazon EC2?
    Opções:
    A. AWS Trusted Advisor
    B. Amazon Inspector
    C. Amazon GuardDuty
    D. AWS Shield
    Resposta correta: B
    Explicação: O Amazon Inspector é um serviço automatizado de avaliação de segurança que ajuda a melhorar a segurança e a conformidade de aplicativos implantados na AWS.

    Pergunta (Escolha 1): Uma empresa precisa criar usuários e grupos de usuários, atribuir políticas de permissões e gerenciar credenciais para acesso à conta AWS. Qual serviço é a solução?

    A. AWS KMS

    Explicação: Uma identidade IAM é uma entidade que pode ser usada para autenticar e conceder permissões para acessar os serviços da AWS.

    Pergunta (Escolha 2): Quais dos seguintes são responsabilidades do cliente sob o Modelo de Responsabilidade Compartilhada da AWS?

Domínio 3: Tecnologia e Serviços da Nuvem (22 questões)

    Pergunta (Escolha 1): Qual é um dos principais casos de uso para o Amazon S3 (Simple Storage Service)?
    Opções:
    A. Executar aplicativos complexos.
    B. Hospedar um banco de dados relacional.
    C. Armazenar arquivos de objetos, como imagens e vídeos.
    D. Criar uma rede privada virtual.
    Resposta correta: C
    Explicação: O Amazon S3 é um serviço de armazenamento de objetos, ideal para armazenar dados não estruturados, como documentos, imagens, vídeos, logs e backups.

    Pergunta (Escolha 1): Você precisa de um serviço de computação escalável que permita executar código em resposta a eventos, sem gerenciar servidores. Qual serviço é a melhor opção?
    Opções:
    A. Amazon EC2
    B. Amazon S3
    C. AWS Lambda
    D. Amazon RDS
    Resposta correta: C
    Explicação: O AWS Lambda é um serviço de computação sem servidor que executa código em resposta a eventos, sem a necessidade de provisionar ou gerenciar servidores.

    Pergunta (Escolha 1): Qual serviço é uma coleção de servidores virtuais (instâncias) na nuvem da AWS?
    Opções:
    A. Amazon S3
    B. Amazon EC2
    C. Amazon RDS
    D. Amazon VPC
    Resposta correta: B
    Explicação: O Amazon EC2 (Elastic Compute Cloud) fornece capacidade de computação escalável na nuvem na forma de instâncias de máquinas virtuais.

    Pergunta (Escolha 1): Uma empresa precisa de um banco de dados relacional totalmente gerenciado na AWS. Qual serviço deve ser usado?
    Opções:
    A. Amazon Aurora
    B. Amazon DynamoDB
    C. Amazon Redshift
    D. Amazon S3
    Resposta correta: A
    Explicação: O Amazon Aurora é um banco de dados relacional compatível com MySQL e PostgreSQL, que é totalmente gerenciado pela AWS. O DynamoDB é NoSQL, o Redshift é um data warehouse e o S3 é um serviço de armazenamento de objetos.

    Pergunta (Escolha 1): Qual serviço de computação sem servidor permite que você execute contêineres Docker sem gerenciar a infraestrutura subjacente?
    Opções:
    A. Amazon EC2
    B. Amazon ECS
    C. AWS Fargate
    D. Amazon EC2 Container Service
    Resposta correta: C
    Explicação: O AWS Fargate é um motor de computação sem servidor para o Amazon ECS e EKS, permitindo que você execute contêineres sem provisionar, gerenciar ou escalar clusters de instâncias EC2.

    Pergunta (Escolha 1): Qual dos seguintes serviços é um banco de dados NoSQL totalmente gerenciado e não relacional da AWS?
    Opções:
    A. Amazon RDS
    B. Amazon DynamoDB
    C. Amazon Redshift
    D. Amazon Aurora
    Resposta correta: B
    Explicação: O Amazon DynamoDB é um serviço de banco de dados NoSQL rápido, flexível e totalmente gerenciado, projetado para aplicativos de alto desempenho.

    Pergunta (Escolha 1): Você precisa criar uma rede isolada na AWS onde possa implantar seus recursos, como instâncias EC2 e bancos de dados. Qual serviço você usaria?
    Opções:
    A. Amazon Virtual Private Cloud (VPC)
    B. Amazon S3
    C. AWS Lambda
    D. Amazon Route 53
    Resposta correta: A
    Explicação: O Amazon VPC permite que você crie uma rede virtual isolada e defina sua topologia, permitindo que você controle a conectividade e a segurança de seus recursos.

    Pergunta (Escolha 1): Qual serviço da AWS é usado para armazenar dados em cache para melhorar o desempenho de aplicativos?
    Opções:
    A. Amazon DynamoDB
    B. Amazon Redshift
    C. Amazon ElastiCache
    D. Amazon SQS
    Resposta correta: C
    Explicação: O Amazon ElastiCache é um serviço de caching na memória totalmente gerenciado, que melhora o desempenho de aplicativos, permitindo que você recupere informações de caches na memória, em vez de depender de bancos de dados baseados em disco mais lentos.

    Pergunta (Escolha 1): Qual serviço de mensagens na AWS é um serviço de enfileiramento de mensagens totalmente gerenciado, que desacopla componentes de aplicativos?
    Opções:
    A. Amazon SNS
    B. Amazon SQS
    C. Amazon Connect
    D. Amazon Kinesis

    Explicação: O Amazon SQS (Simple Queue Service) é um serviço de fila de mensagens, que permite desacoplar e escalar microsserviços, sistemas distribuídos e aplicativos sem servidor.

    Pergunta (Escolha 1): O que o Amazon CloudFront faz?
    Opções:

    Pergunta (Escolha 1): Qual dos seguintes fatores não afeta o custo de uma instância do Amazon EC2?
    Opções:
    A. O tipo da instância.
    B. A região em que a instância está localizada.
    C. A quantidade de dados processados pelo banco de dados da instância.
    D. O sistema operacional da instância.
    Resposta correta: C
    Explicação: O custo de uma instância EC2 é afetado pelo tipo de instância, a região, o sistema operacional e a forma de pagamento, mas não pela quantidade de dados processados pelo banco de dados, que é um custo separado.
